Colores en CSS3

RGB Hexadecimal Seis Dígitos (#rrggbb). Ejemplo: #00FFFF
RGB Hexadecimal Tres Dígitos (#rgb). Ejemplo: #FF0
RGB Decimal rgb(r,g,b). Ejemplo: rgb(198,226,255).
RGB Porcentaje rgb(%r,%g,%b). Ejemplo: rgb (0%, 20%, 80%).
RGB por nombre. Ejemplo: chocolate.
RGBA con opacidad rgba(r,g,b,a). Ejemplo: rgba(198,226,255,0.25).
RGBA Porcentaje rgba(%r,%g,%b,a). Ejemplo: rgba(0%,20%,80%,.5).
HSL (hue, saturation, lightness). Ejemplo: hsl(270,90%,60%).
HSLA (hue, saturation, lightness, alpha). Ejemplo: hsla(270,90%,60%, 0.5).
propiedad Opacidad. Ejemplo: 0.0, 0.5, 1.0.
